开放数据与市民创新的未来

开放数据的发展已经在公共领域展现了其在创新催化，效率提升，经济加速的能量。承接着著名的中央级开放数据计划例如 Data.gov 以及 白宫的开放政府计划，各地方政府逐步开始任命首席数据官，制定开放数据政策，以及建立开放数据门户来尝试开放数据计划。

虽然还在发展初期，但我们已经看到了开放数据改变我们城市未来的巨大潜能。通过提供清洁的公园，与罪案斗争，发给新公司商业执照，地方政府是最能直接影响人民群众生活质量的。也正因为此，在地方一级才有巨大的机遇来利用开放数据重新定义与建立人民与政府的关系。

Beyond Transparency 是一本对开放数据各领域全方位的普查报告，涵盖了前线工作者分享的成功利用开放数据的案例。本书寻求超越为了透明而透明的传统诉求，而号召开始行动与解决实际的问题。通过书中分享的案例，我们可以了解如何建立一个开放数据的生态环境，使开放数据成为有效决策和高效社会服务，超能经济发展，公民权利保障以及公民社会发展的基石。

关于这本书

这是一本由实践者为实践者而编写的书籍。实践者们来自政府内外：既有政府的首席信息官，也有社区领导者更有草根创业者。 Beyond Transparency 旨在记录与传播过往四年里社区在开放数据发展中总结的经验与教训。而我们也知道，社区还在不停的学习。也正因为此，除了提供在亚马逊上售卖的印刷版，我们也将电子版图书以知识共享协议开放给公众阅读与进一步编辑。本书的全文都将开源在 GitHub ，这意味着任何人都可以提交一个新的更新，作出一处修订，乃至与我们一同准备下一版的Beyond Transparency 。现在就来提交你的Pull Request 吧。

Code for America 是美国一个的非营利机构，其旨在建立一个21世纪的由人民创建，为人民服务的政府。在过去4年里，CfA已经与多个城市合作以开放数据为基础支持市民创新。你可以通过 GitHub 来向本书作出贡献。你也可以加入我们的志愿团体，向我们捐助，或者让你的城市加入我们的计划来支持我们。
